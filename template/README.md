# khaos-{{name}}

A Khaos template for starting a new {{instanceName}} quickly.

## Install

Save the template locally with:

```
$ khaos install {{owner}}/khaos-{{name}} {{name}}
```

## Usage

Create a new {{instanceName}} with:

```
$ khaos create {{name}} my-{{name}}
```
