# khaos-khaos

A Khaos template for starting a new Khaos template quickly.

## Install

Save the template locally with:

```
$ khaos install fabiospampinato/khaos-khaos khaos
```

## Usage

Create a new Khaos template with:

```
$ khaos create khaos my-template
```
